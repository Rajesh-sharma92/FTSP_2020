NOTE :- We have 2 types of CSS Selectors like below. 1) Basic CSS Selectors 2) Advanced CSS Selectors 1) Basic CSS Selectors :- i) Element Selectors ii) ID Selectors iii) Class Selectors i) Element Selectors :- To define styling for every Instances of specified Element then we should go for Element Selectors. Ex :- h1{ color : red; } li{ color:blue; } 2) ID Selectors :- styles are applicable only for matched ID Elements. It starts with "#" symbol. EX :- # idname { color : blue; } 3) Class Selectors :- It starts with "." symbol. Ex :- .classname{ color:red; } 2) Advanced CSS Selectors :- ------------------------------- i) * Selectors ii) Descendent Selectors iii) Adjecent Selectors iv) Attribute Selectors v) nth Type of Selectors i) * Selectors :- * Selectors , It means Everything. Ex :- *{ color : blue; } ii) Descendent Selectors :- It means Child tags. Here every "li" tag is perent & every "a" tag is Child Tags only. Ex :- Here we shoud use the " space " symbol. li a{ color:orange; } ul li a{ color:orange; } iii) Adjecent Selectors :- It means side by side. Here we shoud use " + " symbol. Ex :- a+ul { color:red; background: yellow; } iv) Attribute Selectors :- Like 'a' anchor tag is called Attribute kind of Tags. Ex :- Facebook.com a[href]{ color:red; } Ex :- input[type="password"]{ background:red; } v) nth Type of Selectors :- Ex :- li : nth-of-type(even){ color:white; background:red; } Ex :- ul:nth-of-type(even){ color:cyan; background:lightblue; }
