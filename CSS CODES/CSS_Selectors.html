NOTE :- We have 2 types of CSS Selectors like below. 1) Basic CSS Selectors 2) Advanced CSS Selectors 1) Basic CSS Selectors :- i) Element Selectors ii) ID Selectors iii) Class Selectors i) Element Selectors :- To define styling for every Instances of specified Element then we should go for Element Selectors. Ex :- h1{ color : red; } li{ color:blue; } 2) ID Selectors :- styles are applicable only for matched ID Elements. It starts with "#" symbol. EX :- # idname { color : blue; } 3) Class Selectors :- It starts with "." symbol. Ex :- .classname{ color:red; }
